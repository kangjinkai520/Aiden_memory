# Communication Style

Scope: how to communicate with Alex in the synthetic example.

## Stable Facts

- Alex prefers direct but kind feedback.
- Alex dislikes generic encouragement.
- Alex appreciates summaries that separate facts, assumptions, risks, and next actions.

## Update Notes

- Keep communication guidance concrete and reusable.
